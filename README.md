# 🚀 Atividades Flet - Programação Mobile

Este repositório reúne atividades práticas desenvolvidas em **Python** com o framework **Flet**, explorando conceitos de **interface gráfica**, **lógica** e **interação**.

Cada atividade possui **prints ilustrativos** e uma **explicação detalhada** do funcionamento e das interações possíveis.

---

## 📌 Atividades

### 1. 👋 Primeiro App (`1_primeiro_app.py`)
**Descrição:**  
Página inicial que exibe uma mensagem de boas-vindas centralizada.  
Demonstra como configurar título, alinhamento e adicionar textos com diferentes estilos.

**Funcionamento:**
- Exibe um texto grande roxo e mensagens de boas-vindas.
- Sem interação, apenas introdução ao Flet.

- Nome muito curto → `"Nome muito curso"` (laranja)  
- Nome válido → `"✅ Olá, [nome]! Prazer em conhecê-lo(a)!"` (verde)

### 5a. 🧞‍♂️ Desafio 1 - Criador de Perfil (`5a_desafio1.py`)
**Descrição:**  
Formulário para criar um perfil de usuário com validação.

**Funcionamento:**
- Preencher: nome, idade, escola/universidade, hobby.  
- Se houver erro → mensagem em vermelho.  
- Se correto → exibe cartão com perfil (nome, idade, categoria, escola e hobby).  
- Botão **"Excluir"** apaga o cartão.
- Botão **"Editar"** edita as informaçoes do perfil.

### 10. 📱 App Multi-página (`10_app_multipagina.py`)
**Descrição:**  
App com navegação entre Home, Perfil, Configurações, Sobre e Estatísticas.

**Funcionamento:**
- **Home:** boas-vindas e botão estatísticas.  
- **Perfil:** nome, nível, pontos (ganha pontos, nível muda).  
- **Configurações:** modo escuro, notificações, som.  
- **Sobre:** informações do app.  
- **Estatísticas:** mostra pontos/nível em tempo real.  
- Barra de navegação fixa inferior.
